# SBDBT5V_arduino
SBDBT5Vを使って，DualShock３からの値をもとにarduinoを使うコード  
DualShock３からの信号をシリアルモニタで表示させるようにしていますので，目的に合わせて活用してください．  
プログラムのコードが長く煩雑になっていますので，より改善されたものがあればプルリクしてくださるとありがたいです!!

SBDBT5Vの購入サイト：http://www.runele.com/ca1/8/
